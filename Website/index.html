UPI Transaction APIs

Simple APIs to simulate UPI transactions

User Registration API

Register a new user in the system.

{

"username" : "someuser",

"password" : "somesecurepassword",

"phone" : "9876543210"

}

User Login API

Authenticate an existing user.

{

"username" : "someuser",

"password" : "somesecurepassword"

}

Send Money API

Transfer money between users.

{

"sender" : "someuser",

"receiver" : "anotheruser",

"amount" : 500.0,

"otp" : "123456"

}

Check Balance API

Retrieve the user's account balance.

{

"username" : "someuser"

}

View past transactions.
